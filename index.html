Область видимости var

Чему будет равна переменная name?

      
        function test() {
          var name = 'Мария';
        }

        test();
        console.log(name); // ???

Область видимости var

Область видимости var — функция. За пределами функции переменная не доступна.

      
        function test() {
          var name = 'Мария';
        }

        test();
        console.log(name); // undefined

Область видимости var

А теперь чему равна переменная name?

      
        if (true) {
          var name = 'Мария';
        }

        console.log(name); // ???

Область видимости var

Переменная доступна за пределеми блока в котором определена, потому что создание и обращение в одной области видимости.

      
        if (true) {
          var name = 'Мария';
        }

        console.log(name); // Мария

Пример кода

Плохо

      
        var list = [1, 2, 3, 4, 5];
        for (i = 0; i < list.length; ++i) {
          console.log(list[i]);
        }

Хорошо

      
        var list = [1, 2, 3, 4, 5];
        list.forEach(function (item) {
          console.log(item);
        });
